Selenium Demo

This is only a demo so there are limit choices and criteria you can choose from
Requires latest version of Google Chrome

Click download to download the zip file. Extract both folders inside the same directory

Go into the Selenium folder and run SeleniumDemo.jar

A chrome window displaying the JobStreet website and JobStreet form window should be displayed. Fill in the form and click start

After it finishes running (Status=Completed all records) there should be an output file in Desktop/SeleniumDemo

Source Code: https://github.com/113256/SeleniumDemo#seleniumdemo
